# Sueffig & Solide - The Game

Ein kleines Klick-Escape-Game im Proberaum. Es laeuft direkt im Browser und besteht nur aus HTML, CSS und JavaScript.

## Starten

Oeffne `index.html` im Browser.

## Erste Spielversion

- Ein Proberaum-Hintergrund mit vielen Bierflaschen
- Erste Aufgabe: genug gute Biere finden und trinken
- Falsche Bierflaschen zeigen `Ihhhh Schaal`
- Andere Gegenstaende sind erst sinnvoll, wenn ein fetter Beat gefunden wurde
- Das Schlagzeug gibt den Beat erst frei, wenn genug gute Biere intus sind
- Einfaches Inventar
- Grundlage fuer spaetere Song-Raetsel

## Weiterbauen

Gute naechste Schritte:

- nach dem Beat neue Raetsel fuer Gitarre, Amp und Text freischalten
- echte Raetsel fuer Zahlen, Akkorde oder Songteile einbauen
- eingesammelte Gegenstaende noch staerker optisch im Raum verschwinden lassen
- mehr Status im Inventar anzeigen
- eine kleine Songtext-Anzeige als Finale bauen
